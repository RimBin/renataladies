# Video Thumbnails

This folder contains thumbnail images for workout videos.

## Current Placeholders

The following placeholder images are needed:
- `strength-1.jpg` - Full body strength workout
- `cardio-1.jpg` - Cardio interval workout
- `yoga-1.jpg` - Morning yoga flow
- `hiit-1.jpg` - HIIT glutes and legs
- `abs-1.jpg` - 10 min abs workout
- `stretch-1.jpg` - Evening stretching
- `arms-1.jpg` - Arms and shoulders workout
- `dance-1.jpg` - Dance cardio workout

## Image Specifications

- **Aspect Ratio**: 16:9 (e.g., 1280x720px or 1920x1080px)
- **Format**: JPG or WebP
- **Size**: Optimized for web (< 200KB recommended)
- **Content**: Action shot of the workout or instructor in relevant pose

## Using Placeholder Service

Until real images are added, you can use a placeholder service like:
- https://placehold.co/1280x720/F28ACD/FFFFFF?text=Video+Title
- https://via.placeholder.com/1280x720
- Or generate with AI tools like Midjourney, DALL-E

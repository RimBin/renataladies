This is the Next.js public/ folder. Place static assets here and reference them from the root, e.g.:

- /logo.svg
- /images/hero/cover.jpg
- /images/sections/trust-1.jpg
- /images/products/dress-01.webp
- /images/stories/client-01.jpg
- /icons/check.svg
- /video-thumbs/v-01.jpg

Next.js will serve these at / (no extra config needed). For remote images (Unsplash, etc.), configure images.remotePatterns in next.config if you switch to next/image.
